First-time setup

This admin commits posts directly to your GitHub repo using a Personal Access Token. The token lives only in this browser (localStorage), never on a server.

  1. Open GitHub → Fine-grained tokens → New.
  2. Token name: knowjoby blog admin
  3. Expiration: pick what you like (e.g. 90 days).
  4. Repository access: "Only select repositories" → pick knowjoby/knowjoby.github.io.
  5. Repository permissions: set Contents to Read and write. (That's the only permission you need.)
  6. Generate the token, copy it (starts with github_pat_…), and paste below.
